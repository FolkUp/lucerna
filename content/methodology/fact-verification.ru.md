## Обзор

Структурированный процесс верификации утверждений, заявлений и тезисов с использованием разведки на основе открытых источников (OSINT). Разработан для параллельного мультиагентного анализа с целью снижения индивидуальных предубеждений и увеличения охвата.

## Процесс

### 1. Отбор утверждений

Утверждения для верификации отбираются по критериям:
- **Влияние:** на скольких людей это влияет?
- **Виральность:** насколько активно это распространяется / цитируется?
- **Верифицируемость:** можно ли проверить по первичным источникам?
- **Риск:** каковы последствия, если это неправда?

Матрица приоритизации:

| | Высокое влияние | Низкое влияние |
|---|---|---|
| **Высокая виральность** | P0 — верифицировать немедленно | P1 — в очередь |
| **Низкая виральность** | P1 — в очередь | P2 — по мере ресурсов |

### 2. Декомпозиция утверждений

Разбейте сложные утверждения на атомарные, верифицируемые подтверждения:
- Выделите каждое фактическое утверждение отдельно
- Определите неявные допущения
- Зафиксируйте, что должно быть истинным, чтобы утверждение выполнялось

### 3. Параллельное расследование (модель 4 агентов)

Одновременный запуск четырёх независимых исследователей:

| Агент | Роль | Фокус |
|-------|------|-------|
| **Фактчекер** | Верификация утверждений по первичным источникам | Данные, статистика, цитаты, даты, имена |
| **Критик методологии** | Оценка способа получения выводов | Размер выборки, методология, рецензирование, воспроизводимость |
| **Детектор предубеждений** | Выявление конфликтов интересов и фрейминга | Финансирование, аффилиации, систематическая ошибка отбора, умолчания |
| **Исследователь контекста** | Историческая и культурная контекстуализация | Предыстория, связанные события, экспертный консенсус |

Правила:
- Каждый агент работает независимо (без перекрёстного загрязнения)
- Каждый агент формирует структурированный отчёт
- Синтез отчётов — только после завершения всех

### 4. Оценка источников

Для каждого использованного источника — оценка по тесту CRAAP (см. `source-evaluation`):
- Первичный, вторичный или третичный источник?
- К какому уровню (tier) он относится?
- Есть ли подтверждающие источники?

### 5. Оценка уверенности

| Уровень | Критерии |
|---------|----------|
| **Высокий** | 3+ независимых первичных источника согласуются; методология корректна; значительные предубеждения не обнаружены |
| **Средний** | 2 источника согласуются ИЛИ первичный источник с незначительными методологическими замечаниями |
| **Низкий** | Один источник, только вторичные источники или существенные проблемы с методологией / предубеждениями |

### 6. Формат отчёта

```markdown
# Верификация: [Название утверждения]

## Утверждение
[Точная формулировка проверяемого утверждения]

## Источник
[Где появилось утверждение]

## Вердикт
[ИСТИННО / В ОСНОВНОМ ИСТИННО / НЕОДНОЗНАЧНО / В ОСНОВНОМ ЛОЖНО / ЛОЖНО / НЕВЕРИФИЦИРУЕМО]

## Уверенность
[высокая / средняя / низкая]

## Резюме доказательств
[Ключевые находки каждого агента]

## Источники
[Нумерованный список всех проконсультированных источников]

## Ограничения
[Что не удалось верифицировать и почему]
```

## Правила публикации

- Нейтральный тон — представляйте доказательства, а не мнения
- Явно указывайте ограничения
- Не утверждайте большего, чем позволяют доказательства
- Различайте «неверифицировано» и «ложно»
- Корректно указывайте источники

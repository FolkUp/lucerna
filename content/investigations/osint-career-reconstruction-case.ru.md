## Методология

Данное исследование проведено в 3 фазы с использованием исключительно открытых источников (OSINT):

1. **Профилирование** — сбор данных из публичных профилей (GitHub API, техноблог)
2. **Анализ контента** — оценка статей, комментариев, репозиториев
3. **Верификация** — перекрёстная проверка гипотез по независимым источникам

**Источники:** публичные профили, GitHub API, поисковые системы, профессиональные платформы
**Ограничения:** нет доступа к закрытым сообщениям, приватным репозиториям, полной истории. Профиль на LinkedIn не подтверждён.
